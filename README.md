Vehicle Theft Data Analysis (Python)

# Overview

This project analyzes real-world stolen vehicle data to uncover patterns and trends using Python.

# Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Analysis Performed

* Data cleaning (handling missing values & duplicates)
* Exploratory Data Analysis (EDA)
* Statistical testing (T-test, Chi-square)
* Data visualization

# Key Insights

* Station wagons and saloons are the most stolen vehicles
* Most thefts involve vehicles from 2000+
* Neutral colors (silver, white, black) dominate
* Weak correlation between variables suggests multiple influencing factors

# Visualizations

* Top 10 most stolen vehicle types (bar chart)
* Color distribution (horizontal bar chart)
* Correlation heatmap
* Model year distribution (histogram & boxplot)

# Conclusion

Vehicle theft patterns are influenced by usage frequency, availability, and demand.
