# Vehicle Theft Data Analysis (Python)

## Overview

This project analyzes stolen vehicle data to identify patterns and trends using Python.

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Analysis Performed

* Data cleaning (missing values, duplicates)
* Exploratory Data Analysis (EDA)
* Statistical tests (T-test, Chi-square)
* Data visualization

## Key Insights

* Station wagons and saloons are most frequently stolen
* Most stolen vehicles are from year 2000 and above
* Common colors include silver, white, and black
* Weak correlations indicate multiple influencing factors

## Visualizations

* Top 10 most stolen vehicle types (bar chart)
* Color distribution (horizontal bar chart)
* Correlation heatmap
* Model year distribution (histogram and boxplot)

## Conclusion

Vehicle theft patterns depend on multiple factors such as usage and availability.
